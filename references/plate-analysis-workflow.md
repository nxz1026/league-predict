# 盘口背离分析工作流（Plate Analysis）

整合自 football-betting-plate-analysis skill（已合并入 world-cup-predict）。

## 核心原则

**盘口动态变化 > 静态数据（排名/战绩）**

盘口是在用真金白银投票，庄家靠信息优势定价。盘口背离基本面时，优先信盘口。

## 4 步背离分析法

### Step 1：查初始盘口

| 数据 | 来源 | 获取方式 |
|------|------|---------|
| 1X2 赔率 | Oddspedia | `web_extract oddspedia_url` |
| 亚盘 | Oddspedia / Dimers | `web_extract` |
| 大小球线 | Oddspedia / Dimers | `web_extract` |
| 预测市场 | Polymarket | `web_extract`（备用） |

### Step 2：分析盘口合理性

- 排名 vs 实际赔率是否匹配？
- 赔率 ≤ 1.10 → 极端盘口快道（见下方特殊规则）
- 盘口深度与排名的关系

### Step 3：追临场变化

对比初盘 vs 当前盘口：

| 变化方向 | 含义 |
|---------|------|
| 热门方赔率下降/让球减少 | 市场资金集中，真热门 |
| 热门方赔率上升/让球增加 | 冷门信号，庄家防范 |
| 盘口稳定不变 | 信息已被市场消化 |
| 客队水位下降/受让盘口变小 | 市场在压冷门方 |

### Step 4：背离判断

```
盘口 vs 基本面方向一致 → 信任盘口方向 (⭐⭐⭐+)
盘口与基本面背离 → 优先信盘口 (⭐⭐⭐⭐)
盘口模糊/看不清 → 低信心度 (⭐⭐)
```

## 信心度量化标准

| 星级 | 条件 |
|------|------|
| ⭐ | 盘口模糊、基本面也模糊 → 运气球 |
| ⭐⭐ | 单一信号支持（盘口或基本面其中之一） |
| ⭐⭐⭐ | 盘口 + 基本面 2 个信号一致 |
| ⭐⭐⭐⭐ | 盘口 + 基本面 + 临场变化 3 个信号都支持 |
| ⭐⭐⭐⭐⭐ | 盘口 + 基本面 + 临场变化 + 伤停/战意 4 个维度全部支持 |

## 极端盘口特殊规则

热门方赔率 ≤ 1.10（或等价亚盘 ≥ -2），且：
- 盘口深度与赔率强度匹配（如 -3.5 配 1.03）
- 基本面无重大伤停
- 盘口临场无向弱队方向调整

→ 直接 5 星，无需再找额外背离信号。

## 5 项预测定义

| 项目 | 数据来源 | 输出要求 |
|------|---------|---------|
| 方向 + 信心度 | 亚盘 + 1X2 + 基本面 | 推荐方向 + 1-5 星（量化） |
| 比分 | 大小球盘 + 双方进失球 | 具体数字 + 偏差修正 |
| 让球 + 水位 | 亚盘 | 让球方 + 水位参考 |
| 大小球 + 线 | 大小球盘（O/U） | Over/Under + 基准线 |
| 双方进球 (BTTS) | 双方进攻 + 防守 | Yes/No + 依据 |

## 精选踩坑案例

### 1. 弗鲁米嫩塞 1-2 门多萨独立（爆冷）
- ❌ 过于依赖主队排名和杯赛底蕴
- ✅ 初盘 -0.75 水位 -133 偏高 → 本身就是警示信号
- ✅ 客队排名更强 + 5 连胜 → 盘口不信任主队

### 2. 阿森纳 0-0 葡萄牙体育
- ❌ 过于信赖阿森纳主场数据
- ✅ 临场盘口从 -1 跌到 +0.5 → 机构预警主队赢不了

### 3. 巴黎 1-1 阿森纳（欧冠决赛 90min）
- 亚盘 PSG -0.25 极浅盘（PSG 媒体热度 39.68% 但只让平半）
- 客队水位下降 → 资金在压 Arsenal
- Arsenal 9 场零封纪录 vs PSG 44 球进攻 → 盘口 + 基本面一致
- **5/5 全部命中**：方向 ✅ 比分 ✅ 让球 ✅ 大小球 ✅ 双方进球 ✅
- **验证**：盘口 + 基本面背离分析在此场证明有效

### 4. Mexico vs South Africa（揭幕战）
- 亚盘 Mexico -0.5/-1（BetUS/ESPN），但庄家开浅盘而非 -1
- 说明庄家自己在防范爆冷
- Mexico 缺乏稳定得分点 → 保守预测小比分
- **方法论验证**：盘口 + 基本面 + 预测市场三方交叉

### 5. Netherlands vs Japan（身高差+伤停叠加）
- 荷兰均高 6'2" vs 日本 5'10"
- 日本 Endo 退役 + Mitoma 伤退（攻击线双重重创）
- 盘口 -0.5 水位 1.80 轻微支撑荷兰
- **伤病变量 > 静态排名**

- Ecuador（南美预选 18 场失 5 球，对手含巴西/阿根廷）→ 高含金量
