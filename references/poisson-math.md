# Poisson 比分分布 — 设计文档

## 公式

Poisson 联合概率：

```
P(主队进 h 球, 客队进 a 球) = Pois(h; λ_home) × Pois(a; λ_away)
```

其中 `Pois(k; λ) = λ^k × e^{-λ} / k!`

实现采用**对数域计算**避免溢出：

```python
log_p = k * math.log(λ) - λ - math.lgamma(k + 1)
return math.exp(log_p)
```

遍历范围：0..8 球 × 0..8 球 = 81 种比分组合。忽略概率 < 0.1% 的条目。

## λ 计算

```
raw_home = hp × 0.40 + hfs × 0.20 + hrs × 0.15 + sm × 0.25
raw_away = ap × 0.40 + afs × 0.20 + ars × 0.15 + (-sm) × 0.25
raw_draw = dp × 0.50

λ_home = (raw_home + 0.5 × raw_draw) × LAMBDA_MULTIPLIER     # 当前 4.5
λ_away = (raw_away + 0.5 × raw_draw) × LAMBDA_MULTIPLIER
```

### 设计理由

| 设计决策 | 为什么 |
|----------|--------|
| 用 raw_strength 而非归一化概率 | 归一化后强 0.8 vs 弱 0.2 变成 0.6 vs 0.15，λ 差被压缩；原始强度保留绝对差异 |
| 平局强度 50% 分流到双方 λ | 平局概率高的比赛（旗鼓相当）双方预期进球都增加，符合直觉 |
| λ = raw × 4.5 | 强队 raw_strength ~0.7 → λ ≈ 3.2，top-1 比分 3-0。弱队 raw ~0.1 → λ ≈ 0.45，top-1 比分 0-1。λ 不低于 0.3 |
| 封底 λ ≥ 0.3 | 即便最弱队至少有一点点进攻期望，避免全部 0-N 预测 |

## 典型输出

| 场景 | λh, λa | top-3 |
|------|--------|-------|
| 强 vs 弱（巴西 vs 海地） | 3.2, 0.5 | 3-0 (14%) / 2-0 (13%) / 4-0 (11%) |
| 均势（中等 vs 中等） | 1.6, 1.6 | 1-1 (10%) / 1-2 (8%) / 2-1 (8%) |
| 稍强 vs 稍弱 | 2.2, 0.9 | 2-0 (11%) / 2-1 (10%) / 1-0 (9%) |

## 调参

`LAMBDA_MULTIPLIER` 提高 → 所有 λ 增大 → 比分更激进（top-1 包含更多 3-0/4-0）
降低 → 比分偏保守（更多 1-0/1-1）

影响弹性的典型值：
- 4.5 当前平衡点，强队 λ=3.0-3.5，弱队 λ=0.3-0.8
- 3.5 偏保守，强队 λ=2.4-2.8
- 6.0 偏激进，强队 λ=4.0-4.7，可能出现 4-0 主导

## 方向 vs Poisson 不匹配

方向判定和三向加权评分用**归一化后**的概率比较三个方向。
Poisson λ 用**归一化前**的原始强度 + 平局分流。

二者数据同源但数学不同。极接近比赛（方向评 40/30/30）可能 top-1 比分和方向不一致。
**以方向判据为准**，比分分布展示不确定性。
